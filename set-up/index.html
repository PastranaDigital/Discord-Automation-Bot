    Setting up the Bot

    Prerequisites

    • NodeJs
    • npm
    • Node


    Getting the bot online

    1. Clone the repo with the code to your machine
    2. In a terminal navigate to the folder, ex: cd ...\Discord\basicBot\
    3. Run node index.js
    4. If needed, use Ctrl + C to close the terminal and log the bot out
